Формат кода

Шестнадцатиричный код AAAAAAAABBCDDDDDD

  • A метка времени (timestamp в секундах)
  • B кластер
  • C тип
  • D идентификатор пользователя

Все входные данные - целые десятичные числа

Примеры данных для кодирования
